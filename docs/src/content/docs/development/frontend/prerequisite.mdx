## Languages

You should be familiar with TypeScript, HTML and CSS.


## Frameworks

ITELL uses the following frameworks:

- [React](https://react.dev): Familiar with React hooks such as `useState`, `useReducer` and `useContext`, have used a state management library (ITELL uses zustand).

- [Next.js](https://nextjs.org): Familiar with the app router, common data fetching patterns in both server component and client components, server actions, form development.

- [Prisma](https://www.prisma.io/): Know how to use the Prisma ORM to implement database design and crud operations. Currently ITELL uses PostgreSQL.

- [Tailwind CSS](https://tailwindcss.com): Know how to use Tailwind to style React components.
